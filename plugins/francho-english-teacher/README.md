# francho-english-teacher

Use your AI interactions to improve your English.

## What this plugin does

- Adds an English coaching skill that reviews the last user prompt written in English.
- Appends a short score and a brief tip when there is something to improve.

## Skills

- prompt-english-coach: Always runs on English prompts and adds a short correction at the end.

## Usage

- Keep the plugin enabled in your Copilot setup.
- When you write in English, the skill adds a short score and tip in Spanish at the end.
