# francho-python

Python coaching focused on fundamentals, code quality, and Databricks Apps.

## What this plugin does

- Adds a dedicated Python coaching agent for learning and practical guidance.
- Supports explanations, code reviews, refactoring advice, and guided practice.
- Emphasizes clean Python, maintainability, security, and Databricks platform best practices.

## Agents

- Python coach: Helps with Python learning and Databricks Apps development.

## Usage

- Keep the plugin enabled in your Copilot setup.
- Ask for help with Python concepts, code review, refactoring, or practice exercises.
- Use it for Databricks Apps topics such as SDK usage, app structure, and deployment workflows.
